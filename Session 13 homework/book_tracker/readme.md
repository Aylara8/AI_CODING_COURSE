Book Reading Tracker - Homework 2
This application provides a centralized system for users to catalog books they want to read, are currently reading, or have finished. It emphasizes clean code organization, server-side validation, and a responsive user interface.
Full CRUD functionality (Create, Read, Update, Delete).
Data persistence using a structured JSON file system.
AJAX-based deletion for a seamless user experience without page reloads.
Server-side and client-side form validation to ensure data integrity.
Status filtering and real-time library statistics.
