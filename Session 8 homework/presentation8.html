Translation Guide

Same Logic, New Syntax

You already know how to code. You just need to learn the new dialect.

üêç Python
name = "Alice"

def greet(name):
    if name == "Alice":
        print("Hi!")
    else:
        print("Who?")
üìú JavaScript
let name = "Alice";

function greet(name) {
    if (name === "Alice") {
        console.log("Hi!");
    } else {
        console.log("Who?");
    }
}

Notice: Curly braces {} replace indentation. Variables need let.

The Magic Connection

The DOM

Document Object Model. This is how JavaScript grabs HTML elements to change them.

// 1. Select the element
let title = document.getElementById('myTitle');

// 2. Change the text
title.textContent = "Hello World!";

// 3. Change the style
title.style.color = "blue";
